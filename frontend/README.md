# Frontend

Diretório com a interface do usuário, componentes React e páginas Next.js.

## Estrutura

- **app/** - Rotas e páginas Next.js (App Router)
- **components/** - Componentes React reutilizáveis
- **public/** - Assets estáticos (imagens, fontes, etc)
- **jsconfig.json** - Path aliases
- **next.config.mjs** - Configuração Next.js
- **postcss.config.mjs** - Configuração Tailwind CSS

## Scripts

```bash
npm run dev      # Inicia servidor de desenvolvimento
npm run build    # Build para produção
npm start        # Inicia em produção
npm run lint     # Validação de código
```

## Notas

Usa Next.js 16 com App Router e React 19 para máxima modernidade.
